Forwards leads directly to your Lead Assign account

== Description ==

The Lead Assign plugin contains a widget that you can use to send leads directly to your Lead Assign account.

Lead Assign routes leads to the most applicable agents, instantly. Find out more at [Lead Assign](https://leadassign.com).

Contact Form 7 is required to use this plugin.

== Installation ==

1. Install plugin from wordpress.org
2. Activate plugin from plugin screen in Wordpress admin page
3. Configure plugin settings under Settings -> Lead Assign, or through Plugins -> Lead Assign -> Settings
4. Install Contact Form 7 if it is not already installed
5. Add Lead Assign widget to website

== Frequently Asked Questions ==

= Help! Nothing's showing up. What do I do!? =

Make sure that Lead Assign has been configured.
Make sure that the Lead Assign Widget has been added to a widgetarea in Appearance -> Widgets
